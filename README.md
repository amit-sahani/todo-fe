# Todo App (React + Vite)

Simple responsive todo app (no CSS libraries). Data is stored in localStorage.

Features:

- Register / Login (client-only, stored in localStorage)
- Create todos (name, status, priority)
- List todos with pagination (5 per page), search, status filter, sorting (date/priority). Completed items are hidden in list.
- Edit todo in modal
- Activity history with timestamp and filters

Run locally:

1. cd into the folder

```powershell
cd c:/Users/amit/fullstack-todo/todo-fe
npm install
npm run dev
```

Open the Vite URL printed in the terminal.
